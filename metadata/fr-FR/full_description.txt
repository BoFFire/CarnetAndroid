Bien plus qu'une simple application de prise de notes




- Un éditeur complet : gras/italique/soulignement/couleur/surlignage
- Ajout d'images 
- Zones de texte flottantes
- Format HTML
- Organisation hiérarchique des notes par dossier, choisissez vous même le dossier racine en fonction de vos besoins
- Bulle flottante à la facebook pour ré-ouvrir votre dernière note rapidement
- Transparence pour prendre en note à partir d'une application en arrière-plan.
- Quickly see your recent notes
- Recherche 
- Protection via code pin (ne chiffrera cependant pas vos notes)
- Statistiques : mots/phrases/caractères
- Synchronisation Google Drive

Bientôt :
Version Chrome (pour Windows/Mac/Linux/FreeBSD/ChromeOS)
